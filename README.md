# Daily Rituals ✨

A beautiful, mobile-first web app to track and share your daily joys and rituals.

## Features

- **Date Selection**: Pick any day to record your rituals
- **Multi-Select Rituals**: Choose from 4 core activities (coffee, learning, working, fruit)
- **Custom Emojis**: Add your own personal touch with an emoji picker
- **Share**: Share your day with friends via the Web Share API or clipboard

## Design

This app features a warm, organic aesthetic with:
- Distinctive typography (Fraunces + DM Sans)
- Warm gradient backgrounds (peach, coral, terracotta, plum)
- Soft, rounded shapes with generous spacing
- Playful animations and micro-interactions
- Mobile-first responsive design

## Tech Stack

- Pure HTML, CSS, and JavaScript (no frameworks)
- Web Share API for native sharing
- CSS Grid and Flexbox for layouts
- CSS custom properties for theming
- Google Fonts for typography

## Local Development

Simply open `index.html` in your browser. No build process required!
